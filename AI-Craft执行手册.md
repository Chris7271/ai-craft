## 冷启动现实评估

> 本节是对所有策略的诚实审视，在制定执行计划前必须理解这些约束。

### 核心瓶颈：你在内容圈的存在感是零

不管什么策略，零粉丝、零内容圈人脉是最大的现实障碍。技术能力强 ≠ 有人知道你。

### 内容创作是需要学习的技能

这是最容易被低估的问题：

- 写技术文章 ≠ 写社交媒体爆款文案
- 做技术demo ≠ 做吸引人的短视频
- 标题、封面、节奏、文案——这些全是独立技能
- 前20-30条内容大概率数据很差，这不是方向错了，是技能在成长

### 匿名的代价比想象中更大

匿名不只是"不露脸"，它意味着：

- 不能用真实身份背书（PhD、大厂经验），而这恰恰是最大的信任资产
- 不能在本名社交圈分享来获取初始流量
- 信任建立和初始分发的难度翻倍

### 105分钟的产能限制

做一条有质量的结果展示型视频：构思20分钟 + 录屏30分钟 + 剪辑60分钟 + 文案15分钟 = 2小时起步。105分钟只够做一条文字帖或一条粗糙视频。高质量视频实际只能在周六批量生产。

### 真正的时间表

```
Month 1：  ¥0。学习内容创作，发20-30条内容，大部分数据差。产出是经验，不是钱。
Month 2：  ¥0。开始找到感觉，某条内容数据还可以。粉丝可能到200-500。
Month 3：  ¥0-500。如果持续发且有几条不错的内容，可以尝试开知识星球。
Month 4-6：¥500-3000/月。如果方向对，开始有稳定产出和一小批用户。
Month 6+： 有可能月入过万，前提是前5个月没放弃。
```

### 最大的风险不是策略错误

是第一个月没看到结果就放弃了。大多数副业项目死在前3个月，不是因为方向错，是因为预期和现实的落差。

### 快速变现组合策略

虽然"快速变现"不切实际，但可以用组合打法尽量缩短第一笔收入的时间：

```
Week 1-2（立刻启动）：
  ├─ 小红书/B站发"结果展示型"内容（赌算法分发）
  ├─ 即刻/V2EX发有深度的技术观点（建立圈层口碑）
  └─ 如果有可信赖的朋友/前同事有AI需求 → 私下接触（但不要勉强）

Week 3-4（补充渠道）：
  ├─ 如果有内容起量 → 简介挂知识星球，开始转化
  ├─ 如果没起量 → 复盘数据，调整内容方向，继续发
  └─ 筹备第一场Workshop（不急着办，先有100+粉丝再考虑）

核心逻辑：
  不是"先免费做内容 → 攒粉 → 再变现"
  而是"展示结果 → 有人问怎么做 → 引导付费"
  但两条路都需要内容先行，区别是后者更直接。
```

## 二、12周启动执行计划

### Week 0：基建周

```
Day 1（2小时）：确定笔名 + Canva设计Logo/头像 + 做封面模板
Day 2（2小时）：注册小红书/即刻/B站账号，统一资料和简介
Day 3（2小时）：安装OBS + 剪映/DaVinci，做5分钟测试录屏
Day 4（1小时）：创建GitHub教学代码仓库，准备干净的录制开发环境
```

### Week 1-2：小红书密集测试（每天1篇，14天14篇）

选题覆盖6种类型以测试数据：对比实测、热点解析、踩坑实录、反常识、效率暴击、知识卡片。同时即刻每天1条短想法（从小红书提炼核心观点，5分钟额外工作量）。

每篇制作SOP（45-60分钟）：选题确认+写标题(10min) → 准备截图素材(15min) → Canva做6张图(15min) → 写正文+标签(10min) → 晚8-10点发布(5min)。

Week 2结束做数据复盘：哪类选题数据最好、什么标题风格点击率高、什么时间发布效果好、评论区高频问题是什么、关注者画像。

### Week 3-4：内容定型 + B站首发

小红书收敛到数据最好的2-3种内容类型。发布B站第一条独立视频（15-20分钟录屏教程）。

### Week 5-8：Season 1 + 知识星球

开通知识星球(¥199/年，首批50人¥99)。启动Season 1「3天用RAG+Agent重构公司搜索系统」，每周三晚8点发1集，EP01-04公开，EP05-07知识星球独享。

### Week 9-12：Season 2 + Workshop

Season 2「从OpenClaw学Agent架构」。Week 10-11办第一场付费Workshop(¥49，腾讯会议，90分钟live coding)。

### 12周关键里程碑

> ⚠️ 以下指标经过现实校准。零粉丝冷启动的增长速度远低于有基础的账号，前4周主要任务是"学会做内容"而不是"涨粉"。

```
Week 4：  小红书粉丝200-500，B站首条视频播放500-2000
          真正的里程碑：已发布14+篇内容，完成数据复盘，找到1-2个数据相对好的内容方向
Week 8：  小红书500-2000粉，B站200-1000粉，知识星球20-50人，收入¥0-3,000
          真正的里程碑：内容质量明显提升，有稳定的生产节奏，Season 1已开播
Week 12： 小红书1000-5000粉，B站500-3000粉，星球50-150人，累计收入¥2,000-10,000
          真正的里程碑：完成2个Season，有了第一批付费用户和真实反馈
```

**如果实际数据低于以上区间的下限：** 不一定是方向错了，更可能是内容表达方式需要调整。回顾数据最好和最差的5条内容，找规律。

**如果实际数据高于以上区间的上限：** 说明你找到了有效的内容形式，加大投入。

## 每日作息计划

> 核心原则：砍掉所有低效时段，把创作集中在早间和周六；22:00后和周日下午强制休息，保证可持续性。

### 工作日

**早间 · 唯一的创作时段（6:00 - 8:45）**

| 时间 | 动作 |
|------|------|
| 6:00-6:20 | 起床洗漱，不看手机 |
| 6:20-6:30 | 咖啡/茶，打开昨晚备好的大纲 |
| **6:30-8:15** | **深度创作105分钟**——写文/录屏/剪辑，全天唯一也是最好的创作窗口 |
| 8:15-8:45 | 发布或存草稿，记录明天线索，准备出门 |

**早通勤（9:00 - 10:00）· 输入与互动**

- 刷同领域内容，观察什么在爆
- 回复评论（养号权重）
- 收藏素材到备忘录

**10:00 - 21:00 · 上班**

专注本职工作，不碰副业。午休用来真正休息。

**晚通勤（21:00 - 21:45）· 低耗准备**

- 为明早做准备：确定选题、在脑中/备忘录列大纲
- 或听AI相关播客保持输入
- 累了就纯放空，不强求

**到家后（22:00 - ）· 休息**

洗漱放松，自由时间，不工作。23:00-23:30 入睡，支撑次日 6:00 起床。

### 周末

**周六 · 生产日**

| 时间 | 动作 |
|------|------|
| 8:00-12:00 | 批量创作（4小时深度，产出3-4篇内容或1-2个教程视频） |
| 14:00-16:00 | 剪辑/排版/封面/预设定时发布 |
| 16:00-17:00 | 安排下周发布计划 + 社群运营集中处理 |

**周日 · 复盘 + 强制休息**

| 时间 | 动作 |
|------|------|
| 9:00-10:30 | 周复盘：数据、反馈、策略微调 |
| 10:30-11:30 | 列下周5天选题 + 每天的大纲骨架 |
| **11:30之后** | **强制休息，不碰项目** |

### 周度时间预算

| 类别 | 工作日 | 周末 | 合计 |
|------|--------|------|------|
| 深度创作 | 8.75h（早间105min×5） | 6h（周六） | ~15h |
| 输入/素材 | 5h（通勤） | 0 | ~5h |
| 规划/复盘 | 3.75h（晚通勤备题） | 2.5h（周日） | ~6h |
| **总计** | | | **~22h/周** |

### 精力保障

**睡眠**

- 固定23:00-23:30入睡，不管周末还是工作日，生物钟稳定后6:00自然醒
- 到家后不看短视频、不刷信息流——大脑兴奋会延迟入睡
- 某天没睡好，允许跳过当天早创作，不硬撑出低质量内容

**起床后前10分钟**

- 不看手机——刷消息会消耗意志力和注意力预算
- 一杯咖啡或茶作为固定启动仪式，身体形成条件反射
- 直接打开昨晚的大纲，零决策启动

**创作环境**

- 固定一个位置只做创作，大脑会把空间和"产出模式"关联
- 手机放到另一个房间或开勿扰
- 需要背景声用白噪音，不开播客

**身体层面**

- 前一天晚饭别太晚太重，消化负担影响睡眠和次日早晨状态
- 起床后先喝一杯水再喝咖啡
- 起床后5分钟简单拉伸，不需要健身，只是唤醒身体

### 饮品与早餐

> 目标：支撑6:30-8:15的105分钟高强度脑力输出，前晚全部准备好，早上零操作。

**饮品：咖啡不是必须的**

关键是固定的启动仪式，不是咖啡本身。咖啡、茶、热水都行，每天重复同一个动作让身体形成条件反射。

- 如果喝咖啡：起床后先喝水，6:20左右喝咖啡效果最好（皮质醇在起床后30-60分钟下降，此时咖啡因接棒），不要空腹喝
- 如果不想依赖咖啡：绿茶/红茶完全够用，咖啡因更低但释放更平缓
- 睡足+固定作息稳定后，有些人不需要任何咖啡因

**早餐时间：6:15-6:25吃完，创作前解决**

105分钟是长时段，空腹到后半段血糖掉下来会走神。吃完留5分钟缓冲再开工。

**推荐方案（高蛋白+适量脂肪+低糖，前晚备好）：**

| 方案 | 准备时间 | 做法 |
|------|---------|------|
| 隔夜燕麦 | 0分钟 | 前晚：燕麦+牛奶+奇亚籽+坚果放冰箱，早上直接吃。周日做5份 |
| 水煮蛋+全麦面包 | 0分钟 | 前晚煮好蛋放冰箱，面包常备，早上冷吃或微波30秒 |
| 希腊酸奶碗 | 1分钟 | 酸奶+坚果+蓝莓倒一起，蛋白质高 |
| 杂粮粥+鸡蛋 | 0分钟 | 电饭煲预约定时6:00好，配前晚煮好的蛋 |
| 豆浆+茶叶蛋+红薯 | 0分钟 | 豆浆机预约，红薯前晚蒸好，都是慢碳水 |

**避开：** 包子油条烧饼（高油高碳水，吃完昏沉）、甜麦片/果汁（本质是糖，血糖过山车）、只喝牛奶或只吃水果（蛋白质不够，撑不到8:15）。

**最省事方案：固定一个菜单每天吃一样的。** 周日晚上做5份隔夜燕麦放冰箱，工作日零准备时间，决策精力留给创作。

### 三条纪律

1. **晚通勤定好明天选题和大纲** → 早上坐下来直接写，零启动成本
2. **22:00后和周日下午不碰项目** → 睡眠和休息是6点起床的前提
3. **没睡好就跳过早创作** → 可持续比单日产出重要

## 三、私域运营设计

### 为什么需要私域

公域流量（小红书/B站）是别人的，算法随时变；私域流量（微信/星球）是自己的，可反复触达。

```
公域的作用：获取新用户（拉新）
私域的作用：留住老用户（复购+转化高价产品）
```

### 私域链路

```
小红书/B站（免费内容，触达10万+人）
       ↓ 个人简介引导
知识星球（¥199/年，筛选付费用户）
       ↓ 星球内引导添加微信
微信个人号（1对1关系，最深信任）
       ↓ 标签分组管理
朋友圈 + 微信群（精准推送高价产品）
```

### 知识星球运营SOP

```
日常更新（每周2-3条）：
  · 周一：本周计划 + 一条深度技术思考
  · 周四：视频/文章的补充代码和深度解析
  · 不定期：回答用户提问（承诺48小时内回复）

内容分层：
  公开平台给 → "是什么"和"为什么"
  知识星球给 → "怎么做"的完整细节 + 源代码 + 个人经验判断

续费策略：
  · 年底前集中发布"年度精华汇总"
  · 续费期前1个月发布重磅内容（新Season付费集或独家技术报告）
  · 老用户续费价不变，新用户逐步涨价（制造"早加入更划算"心理）
```

### 微信个人号运营

```
添加时机：
  用户在知识星球提了一个好问题 → 回答后说"加我微信，这类问题我们可以聊更多"
  不要无差别引导所有人加微信——只加高质量/高意向用户

标签分组：
  · A类：技术强、活跃度高、可能买课/咨询 → 重点维护
  · B类：普通付费用户 → 正常触达
  · C类：潜在企业客户 → 单独跟进

朋友圈策略（每周3-5条）：
  · 40%：技术内容片段（"今天踩了个坑……"），展示你在一线做事
  · 30%：产品/内容预告（"下周的Season主题是……"），制造期待
  · 20%：个人思考（"关于AI Agent落地，我有几个判断……"），塑造专家感
  · 10%：轻度互动（投票、提问），维持活跃度
  · 绝不刷屏，绝不硬广
```

### 微信群（阶段四再做）

前12个月不开群。原因：群运营太重，且小规模时群活跃度难以维持。等知识星球超过1000人后，再考虑开设"AI Craft核心群"（限星球老用户加入，控制质量）。

### 私域转化高价产品的路径

```
微信好友看到你的朋友圈 → 发现你在做系统课程
  → 因为已经通过星球建立了信任
  → 购买课程的决策成本大幅降低

同理：
  → 发现你开始接咨询
  → 因为长期关注知道你水平
  → 主动询价的概率远高于陌生用户

私域的本质不是"运营"，是"让信任自然发酵"。
```

## 五、定价心理学

### 核心原则

```
定价不是"产品值多少钱"，而是"用户愿意在什么心理状态下付多少钱"。
```

### 价格锚点设计

```
免费内容（小红书/B站）
  ↓ 用户心理："这些免费内容就已经这么好了"
  ↓ 锚点效应：免费内容建立了"超值"预期
知识星球 ¥199/年
  ↓ 用户心理："免费的都那么好，付费的肯定更好"
  ↓ 对比锚点：一顿火锅的钱 / 一本技术书的钱
  ↓ 换算话术："每天5毛钱，跟一个AI博士学Agent"
系统课程 ¥999
  ↓ 用户心理：已在星球里见识了你的水平
  ↓ 对比锚点：极客时间类似课程卖2000-5000
  ↓ 换算话术："不到一次面试辅导的钱，学完能直接写进简历"
Workshop ¥49
  ↓ 定位：不是利润产品，是信任建立产品
  ↓ 用户心理："49块看他live coding，值不值试一试"
  ↓ 目的：体验过live coding后，转化到星球和课程
```

### 定价策略

**知识星球：¥199/年（锚定）**

```
首批50人：¥99（早鸟价）
  → 目的：快速积累种子用户和口碑
  → 心理：用户觉得"赚到了"，愿意主动传播

50-200人：¥149
  → 制造"早加入更便宜"的紧迫感
  → 在小红书/B站公开说"即将涨价"

200人以后：¥199（标准价）
  → 不再打折，建立价格认知

续费：老用户永远按首次加入时的价格续费
  → 奖励忠诚度，降低流失率
```

**系统课程：¥999（核心利润品）**

```
首发定价：¥699（限时2周）
  → 利用"预售+限时"制造紧迫感
  → 首发期集中销售，制造"很多人在买"的社会证明

正式定价：¥999
  → 这个价位的心理意义：三位数和四位数的分界线
  → ¥999感觉"不到1000"，心理上属于"中等投入"
  → 对比锚点：极客时间类似课程2000-5000元

课程组合价：
  · 课程 + 知识星球年费 = ¥1099（省¥99）
  → 用户觉得"星球白送了"
  → 实际上你同时锁定了一个年费用户
```

**涨价策略**

```
每门课程每6个月提价一次（¥100-200）
理由："内容持续更新，新增了X个模块"

涨价前1周：
  · 小红书/即刻/B站同步发布"涨价预告"
  · 知识星球内提前2周通知
```

```
企业内训：¥30,000/半天 起步
  → 不主动推销，等品牌溢出后自然有人找来
  → 报价时附上你的背景、粉丝数、课程销量作为背书
```

## 六、增长黑客战术

### 0→1000粉的冷启动破局

冷启动是最难的阶段。没有粉丝，算法不推你；算法不推你，就没有粉丝。以下是具体的破局手段：

#### 战术1：评论区截流

```
操作：
  找到你所在领域的大V（李沐、秋芝2046、AI相关热门视频）
  在他们的视频/笔记下留高质量评论
  不是"写得好"这种废话，而是：
    · 补充一个他没提到的技术细节
    · 指出一个常见的误区
    · 分享自己的实战经验

效果：
  大V的评论区就是流量池
  一条高赞评论可以带来50-200个主页访问
  如果你的主页内容够好，10-20%会关注你

频率：每天花10分钟，在3-5条热门内容下留高质量评论
```

#### 战术2：蹭热点的正确姿势

```
AI领域几乎每周都有大事件（新模型发布、产品更新、安全事件等）

操作：
  事件发生后2-4小时内发布你的专业解读
  不是搬运新闻，而是加上你的专业判断
  格式："[事件] + 作为一个做了10年算法的人，我的3点看法"

关键：速度。热点内容在事件后4小时内发布流量最大，24小时后衰减80%。

实操：
  · 关注Twitter/X上的AI大V（@sama, @AnthropicAI等）
  · 关注即刻"AI探索站"圈子
  · 设置Google Alerts关键词：AI Agent, OpenAI, Anthropic, DeepSeek
  · 有大事件 → 午休或下班后立即写一篇发出去
```

#### 战术3：小红书SEO

```
小红书的搜索流量是长尾的——一篇好内容可以持续被搜索到几个月。

  · 在标题和正文中自然嵌入这些关键词

长尾词举例：
  大词：#AI #人工智能 #编程
  中词：#AIAgent #RAG #大模型开发
  长尾：#Agent开发教程 #RAG搜索优化 #MCP协议
```

```
  · 在V2EX/Ruby China等技术论坛的相关讨论中贡献有价值的回答
```

## 八、风险预案

### 风险1：被公司发现

```
预防措施：
  · 笔名 + 不提公司 + 内容不涉及公司业务
  · 不在工作时间发内容（定时发布）
  · 朋友圈对同事分组不可见
  · 不用真人照片/声纹可被AI识别的风险低但存在

如果被发现：
  场景A：同事发现但没上报
    → 坦诚但低调："业余做点技术分享"
    → 强调不涉及公司业务
    → 大多数情况下同事不会深究

  场景B：上级或HR找谈话
    → 准备好的说辞："这是个人技术博客，内容都是公开知识"
    → 检查劳动合同：如果没有竞业条款和兼职禁止条款，你有权利做
    → 如果有相关条款：评估风险，可能需要低调一段时间或协商
    → 最坏情况：如果副业收入已达到工资1.5倍，这反而可能是离职的契机

  关键：永远不要在内容中暴露公司信息，这是红线。
```

### 风险2：内容被抄袭/搬运

```
预防措施：
  · 每条视频加品牌水印
  · 代码仓库使用MIT License但要求署名
  · 在视频/图文中自然口播品牌名（搬运者无法去掉语音中的品牌植入）

如果被抄袭：
  · 小规模搬运：忽略。花时间维权不如花时间生产新内容。
  · 大规模搬运：平台举报（B站/小红书都有原创保护机制）
  · 真正的护城河不是内容本身，而是你持续产出的能力和个人IP。
    抄袭者能抄你的一条内容，抄不了你的更新速度和深度。
```

### 风险3：内容翻车（技术错误/争议言论）

```
预防措施：
  · 每篇技术内容中的代码确保能跑通
  · 不确定的地方诚实说"这是我的理解，可能有误"
  · 避免对具体公司/产品做过于绝对的负面评价

如果翻车：
  · 技术错误：立即发更正帖，诚恳道歉，补充正确信息
    → "之前那篇关于XXX的帖子，有一个错误，更正如下"
    → 诚恳纠错反而能加分——用户会觉得"这人很实在"
  · 争议言论：不删帖，不狡辩，补充完整的论证
    → 技术社区尊重"有理有据地坚持观点"，鄙视"说错了就删帖装没事"
```

### 风险4：持续投入但数据始终不好

```
诊断检查清单（如果3个月后粉丝<1000）：

  □ 是不是内容方向不对？
    → 看互动最好的3篇和最差的3篇，找规律
    → 可能你认为有价值的方向用户不买账

  □ 是不是表达方式不对？
    → 找3个非技术朋友看你的内容，问"看得懂吗？想看更多吗？"
    → 知识诅咒是最常见的问题

  □ 是不是平台选错了？
    → 如果小红书数据差但即刻互动好 → 可能你的内容更适合圈层平台
    → 调整主力平台

  □ 是不是发布时间/频率不对？
    → 试试不同时间段发布
    → 如果一天1篇压力太大导致质量下降 → 减到3篇/周但提高质量

  □ 是不是赛道天花板就这么高？
    → 不太可能。AI Agent赛道热度在上升期。
    → 如果你的内容真的有深度但就是没人看——大概率是表达/包装问题，不是方向问题

止损线：
  如果6个月后月收入仍<¥2000 且看不到增长趋势
  → 认真考虑是否继续，或做重大方向调整
  → 但不要在3个月内就放弃——冷启动期天然痛苦
```

### 风险5：精力不足/倦怠

```
预防：
  · 严格控制在每周12小时以内
  · 批量生产（周末集中做），工作日只做发布和互动
  · 给自己每月1周的"轻量周"——只发即刻短想法，不做视频
  · 不追求日更——如果日更导致质量下降，改为隔天更

如果感到倦怠：
  · 暂停1-2周。真的停。不发任何内容。
  · 用户不会因为你停更2周就取关——但会因为你持续发低质量内容而取关
  · 恢复时发一篇诚实的复盘："停更了2周，原因是……"（这本身就是好内容）
```

```
─────────────────────────────────────────
Year 1 累计：¥20,000-60,000（中性约¥40,000）

对比之前版本：原预测Year 1中性¥250,000，下调至¥40,000。
原因：前3个月零收入、零粉丝转化率低、首场Workshop规模小。
```

**最大概率事件：前3个月总收入为零。** 这不是失败信号，是冷启动的正常代价。

### 盈亏平衡分析

```
月固定成本：~¥260
时间机会成本：每周10-12小时 × 4周 = 40-48小时/月

如果按时薪¥200估算机会成本（保守）：
  月机会成本：48 × 200 = ¥9,600

真正的盈亏平衡 = 月收入 > ¥10,000
预计达到时间：Month 10-12（中性情景）

诚实地说：Year 1大概率无法覆盖机会成本。
但这笔账不能只算短期——积累的品牌、内容、技能是长期资产。
```

### 离职决策的财务标准

```
满足以下全部条件时可以考虑离职：

  □ 副业月收入连续3个月 > 工资的1.5倍
  □ 收入来源至少有3个（星球+课程+咨询/Workshop）
  □ 有至少6个月的生活费储蓄
  □ 收入趋势向上（不是偶尔的高峰）

按中性情景，这个时间点大约在 Year 2 末到 Year 3 初。
之前预测Year 2 Month 6-8，过于乐观。
```

### 现金流特征

```
注意：知识付费的现金流有"波峰波谷"特征：

波峰（收入高的时期）：
  · 新课程首发期（集中销售）
  · 知识星球年底续费期
  · 热点事件后的流量高峰

波谷（收入低的时期）：
  · 两个Season之间的空档期
  · 节假日（用户学习意愿下降）
  · 没有新产品发布的平淡期

应对：
  · 预留2-3个月的运营资金
  · 尽量让产品发布和续费期错开
  · 持续有免费内容流量保底
```

```
时机：粉丝达到5000+后再开始互推（太早没人理你）

寻找互推对象的标准：
  · 粉丝量和你在同一个数量级（0.5x-2x）
  · 内容领域相关但不完全重叠
  · 如：你做AI Agent工程 + 对方做AI产品设计/创业
  · 互相补充而非互相竞争

互推方式：
  方式A：互相在视频/帖子中提到对方
    "关于AI产品设计的部分，推荐看XX的内容，他比我专业"

  方式B：联合内容
    一起做一个对话/讨论视频

    你去对方的知识星球做一次分享，对方来你的星球做一次

  1. 先关注目标博主，在他们的内容下留高质量评论（1-2周）
```

## 十一、法律与税务

### 个人收入纳税

```
知识星球/课程收入属于"劳务报酬"或"经营所得"，需要纳税。

劳务报酬税率：
  · 每次收入≤4000元：减除费用800元后按20%
  · 每次收入>4000元：减除20%费用后按20%-40%

实操建议：
  · Year 1收入较低时：按劳务报酬申报，大部分平台会代扣代缴
  · 知识星球：平台代扣代缴个人所得税
  · 自有渠道收入（如课程直销）：需要自行申报
  · 建议找一个兼职会计或用个税APP自行申报

重要：不要逃税。金额小的时候没人查，金额大了一定会查。从一开始就养成合规习惯。
```

### 什么时候注册公司

```
不需要注册公司的阶段：
  · 年收入 < 50万
  · 只有个人的内容和课程收入
  · 不需要开发票

需要注册公司的信号：
  · 企业客户需要开发票（企业内训必须有发票）
  · 年收入 > 50万（个人所得税税率过高，公司税务更优化）
  · 需要签正式合同（版权授权、平台合作等）
  · 要雇人（哪怕是兼职）

注册类型建议：
  · 个体工商户（最简单，适合年收入50-200万）
  · 有限责任公司（适合年收入200万+，有团队，需要融资可能）

时间点：大约在阶段三末期到阶段四初期（Month 12-18）
```

### 知识产权保护

```
你的内容本身受著作权法保护（创作完成即有版权），但以下情况需要额外注意：

课程内容：
  · 在课程页面明确声明版权归属
  · "本课程内容版权归AI Craft所有，禁止未经授权的录屏、转载和分发"
  · 技术上：视频加水印，PDF加个人信息水印

品牌保护：
  · 当品牌有一定知名度后（粉丝>5万），考虑注册"AI Craft"商标
  · 商标注册费用：约¥300-1000/类（可自行在国家商标局网站申请）
  · 优先注册第41类（教育培训）和第42类（科学技术服务）

代码仓库：
  · 公开代码使用MIT License（允许他人使用但要求署名）
  · 课程配套代码可以用更严格的License（仅限个人学习使用）
```

## 十二、AI辅助内容生产

### 核心理念

```
你是做AI教育的，如果自己不用AI来提升生产效率，既是资源浪费也缺乏说服力。
"吃自己的狗粮"本身就是最好的内容素材。
```

### 具体用法

#### 选题和策划

```
工具：Claude / ChatGPT
用法：
  "我在小红书做AI Agent实战内容，目标用户是中级开发者。
   以下是我最近5篇数据最好的选题：[列出]
   帮我生成20个类似风格的新选题，要求：
   1. 标题有冲击力
   2. 从痛点出发
   3. 和Agent/RAG/MCP相关"

注意：AI生成的选题需要你用专业判断筛选。大概20个里能用5-8个。
```

#### 文案撰写

```
工具：Claude
用法：
  把你录屏时的语音转文字（用剪映自动字幕导出），然后：
  "以下是我一个技术视频的口述内容，帮我改写成小红书笔记风格：
   - 300字以内
   - 口语化，像和朋友聊天
   - 每段不超过2行
   - 结尾加一个引导互动的问题"

效率提升：文案撰写从15分钟压缩到5分钟（AI写初稿，你改3分钟）
```

#### 代码文档生成

```
工具：Claude Code / Cursor
用法：
  · 用AI自动生成代码注释和README
  · 用AI生成API文档
  · 用AI生成"快速开始"教程

  这些文档直接放到知识星球和GitHub，提升付费内容的价值感。
```

#### 视频字幕和多语言

```
工具：剪映（自动字幕） + AI翻译
用法：
  · 剪映一键生成中文字幕（准确率95%+）
  · 如果要做英文字幕（触达海外用户）：用AI翻译后人工校对
  · 字幕不只是辅助功能——很多人在公共场合静音看视频，字幕是必须的
```

#### 数据分析

```
工具：Claude
用法：
  每周把各平台数据整理成表格，让AI分析：
  "以下是我过去一周在小红书的发布数据：[表格]
   帮我分析：
   1. 哪类内容表现最好？
   2. 有什么数据异常？
   3. 下周应该重点做哪个方向？"

效率提升：数据复盘从30分钟压缩到10分钟
```

#### 内容二次加工

```
一条B站视频 → AI自动生成：
  · 3个不同角度的小红书标题
  · 5张内容卡片的文案
  · 2条即刻短想法
  · 1篇知识星球深度补充
  · 1篇掘金技术文章

工具：Claude
Prompt：
  "以下是我一个15分钟技术视频的完整字幕文本：[粘贴]
   请帮我生成以下内容，每个独立输出：
   1. 3个小红书标题（痛点驱动，有数字，<20字）
   2. 5张内容卡片的文案（每张50-80字，提炼核心观点）
   3. 2条即刻短想法（100-200字，观点鲜明）
   4. 1篇知识星球补充（500字，补充视频里没展开的技术细节）"

效率提升：跨平台内容改编从2小时压缩到30分钟
```

### "吃狗粮"作为内容素材

```
每次你用AI辅助完成了一项工作，这本身就可以变成一篇内容：

  "我用Claude帮我生成了一周的小红书选题，效果怎么样？实测"
  "AI帮我写代码文档，我打几分？逐行评审"
  "用AI做内容二次加工，我的完整workflow分享"

真实使用AI → 产出使用体验内容 → 内容吸引用户 → 用户付费学你怎么用AI
这就是飞轮。
```
